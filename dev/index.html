Optimization.jl: A Unified Optimization Package

Optimization.jl is a package with a scope that is beyond your normal global optimization package. Optimization.jl seeks to bring together all the optimization packages it can find, local and global, into one unified Julia interface. This means, you learn one package, and you learn them all! Optimization.jl adds a few high-level features, such as integrating with automatic differentiation, to make its usage fairly simple for most cases, while allowing all the options in a single unified interface.

Installation

Assuming that you already have Julia correctly installed, it suffices to import Optimization.jl in the standard way:

import Pkg
Pkg.add("Optimization")

The packages relevant to the core functionality of Optimization.jl will be imported accordingly and, in most cases, you do not have to worry about the manual installation of dependencies. However, you will need to add the specific optimizer packages.
